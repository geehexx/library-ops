# RPG PRD Template

Use this only as a local structure reference. The canonical project PRD is `.taskmaster/docs/prd.md`.

## Required sections

1. Overview, success metrics, and non-goals.
2. Source-of-truth/governance model.
3. Standards profile.
4. Socratic ambiguity log.
5. Capability tree with `### Capability:` headings.
6. Features with `#### Feature:` headings.
7. Data model and architecture views.
8. Testing, quality gates, security, deployment, and risks.
9. Task Master parsing contract.
10. Evidence/source register.

Task Master should treat capability headings as top-level task candidates and feature headings as subtasks.
